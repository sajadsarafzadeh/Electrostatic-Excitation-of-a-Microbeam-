# Simulation of an electrostatically excited microbeam

## Project Overview
This project presents a numerical simulation of the Resistance Spot Welding process using **COMSOL Multiphysics**. The study utilizes a 2D-axisymmetric model to predict temperature distribution and nugget formation between two steel sheets.

## Technical Details
* **Physics Coupled**: Electric Currents (AC/DC) and Heat Transfer in Solids.
* **Model Geometry**: Includes copper electrodes, water-cooling channels, and steel workpieces.
* **Key Analysis**: Transient study of heat generation via Joule heating and the resulting thermal gradients over time.
* **Boundary Conditions**: Implementation of contact resistance and convective cooling for the electrodes.

## Results
The simulation successfully visualizes the peak temperatures at the interface of the sheets, providing insights into the heat-affected zone (HAZ) and the welding current's efficiency.

## Tools Used
* COMSOL Multiphysics
